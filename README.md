# SEO Services Website

Landing page for a company that provides SEO optimization and website promotion services.

The project represents a marketing website where users can learn about SEO services and contact the company.

## Tech Stack

React  
JavaScript  
CSS  

## Features

- marketing landing page
- services section
- contact section
- responsive layout
- modern UI

## Project Structure

src/
components/
pages/
App.js
index.js
